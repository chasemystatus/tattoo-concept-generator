# Tattoo Concept Generator

A simple web app that generates a personalised tattoo concept from a user prompt.

## Live links

- Dev preview (Netlify branch deploy): https://dev--tattoo-concept-generator.netlify.app
- Production (Netlify main site): https://tattoo-concept-generator.netlify.app

## What it does

- User enters a tattoo idea
- App calls the SheCodes AI API
- App displays the generated concept on the page

## Tech used

- HTML, CSS, JavaScript
- Axios
- SheCodes AI API
